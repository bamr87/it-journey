# GH-600 Exam Overview

**Official Name:** Developing in Agentic AI Systems  
**Certification:** GitHub Copilot Specialist  
**Passing Score:** 70% (700/1000)  
**Format:** Multiple choice, multi-select, scenario-based

## Domain Weights

| Domain | Topic | Weight |
|---|---|---|
| D1 | Agentic AI in the SDLC | 18% |
| D2 | Tools, environments, and agent capabilities | 18% |
| D3 | Memory, context, and state management | 19% |
| D4 | Evaluating and improving agent performance | 19% |
| D5 | Multi-agent system design | 17% |
| D6 | Responsible autonomy and guardrails | 9% |

**D3 and D4 are the highest-weighted domains. Prioritise them.**

Official study guide: [Microsoft Learn — GH-600 Study Guide](https://learn.microsoft.com/en-us/credentials/certifications/resources/study-guides/gh-600)
