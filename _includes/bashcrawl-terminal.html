{% comment %} Bashcrawl Web Terminal Embed ============================ Embeds the live, browser-playable Bashcrawl terminal (the interactive web TUI deployed from the bamr87/bashcrawl repository, which is vendored into this site as the `submodules/bashcrawl` git submodule). The game runs entirely client-side and starts in the Entrance chamber, so each quest page passes a `room` label telling the player where to navigate once the dungeon loads. The quest page itself is the walkthrough / strategy guide for that chamber.
