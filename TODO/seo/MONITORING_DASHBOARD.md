## ðŸ” Google Search Console Checklist

### Weekly Review Tasks

- [ ] **Performance Report**
  - Export last 7 days data
  - Compare to previous week
  - Note significant changes (+/- 20%)

- [ ] **Coverage Report**
  - Check for new errors
  - Review excluded pages
  - Verify indexed page count

- [ ] **Core Web Vitals**
  - Check mobile scores
  - Check desktop scores
  - Note any failing URLs

- [ ] **Sitemaps**
  - Verify sitemap status
  - Check submitted vs indexed pages

### Monthly Review Tasks

- [ ] **Keyword Analysis**
  - Export top 100 queries
  - Identify new ranking keywords
  - Find declining keywords
  - Discover content opportunities

- [ ] **Page Analysis**
  - Top 20 pages by impressions
  - Pages with high impressions, low CTR (optimize)
  - Pages with good CTR, low impressions (promote)
